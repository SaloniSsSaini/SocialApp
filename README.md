# 🚀 SocialApp – Full Stack Social Media Platform

A fully functional **MERN Stack Social Media Application** where users can create posts, upload images, like, and comment — deployed live on cloud.

---

# 🌐 Live Demo

🔗 Frontend: https://social-app-35wg.vercel.app
🔗 Backend: https://socialapp-3-ah53.onrender.com

---

# 📌 Features

## 🔐 Authentication

* User Signup & Login
* JWT-based authentication
* Protected routes

## 📝 Post System

* Create post (Text / Image / Both)
* Drag & Drop image upload
* Real-time feed updates

## ❤️ Social Interaction

* Like / Unlike posts
* Comment on posts
* View total likes & comments

## 🎨 UI/UX

* Clean Material UI design
* Dark Mode toggle 🌙
* Responsive layout

## 📦 Advanced Features

* Image upload using Multer
* REST API architecture
* Secure token handling
* Pagination support (backend)

---

# 🛠 Tech Stack

## Frontend

* React.js
* Material UI
* Axios
* React Router DOM

## Backend

* Node.js
* Express.js
* MongoDB Atlas
* Multer (Image Upload)
* JWT Authentication

# 🚀 How to Run Locally

## 🔹 Backend

```bash
cd backend
npm install
node server.js
```

## 🔹 Frontend

```bash
cd frontend
npm install
npm start
```

---

# 🔄 Application Flow

1. User opens the application
2. Signup page appears
3. User creates account
4. Login page opens
5. User logs in
6. Social Feed loads
